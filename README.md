# WdLanguage
This is repository to development language **WD**

***wd language is framework for create aplication with gui(tkinter | ...) and langaege(python | ...)***

# next features
wdlang -> gui => tkinter
wdlang -> manager => pytnon3

wdlang require -> package.json

example:
  package.json
    {
      "gui" : "tkinter"
      "mananger" :  
        "isInterpreted" : "python3"
    }
